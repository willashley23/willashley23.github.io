8-bit Asteroids

JavaScript Canvas

A recreation of the Atari classic, my take on Asteroids combines the retro feel of the original game with a few personal twists added in. The game was written entirly in vanilla JavaScript and Canvas. All of the game logic, including phyiscs for collision detection, vectoring, spawn percentage, and object rotation were all hand-rolled in order to create an authentic 79' throwback.

IdentiFont

JavaScript Chrome Extension

IdentiFont is a chrome extension that saves web developers, UI/UX engineers, and typeface enthusiasts the headache of having to dig through a web page's HTML in order to find that coveted piece of type. Simply hover over a section of text, and IdentiFont will respond with a tooltip indicating the font-family.
